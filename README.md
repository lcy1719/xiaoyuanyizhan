## 【功能介绍】
**本系统包含管理员、员工、用户三个角色，
管理员：管理快递仓库信息、管理待发货信息、管理已收快递、管理物流信息、管理留言信息、管理员工和用户资料。
员工：更改物流信息、管理快递仓库信息、管理待发货信息、管理已收快递、发布留言信息。
用户：签收快递、查看系统公告、发布留言、查看已收快递信息、查看快递物流信息。**
## 【技术选型】
**①：架构: B/S、MVC
②：系统环境：Windows
③：开发环境：IDEA、JDK1.8、Maven、Mysql、
④：技术栈：spring + mybatis + springmvc + jsp**
